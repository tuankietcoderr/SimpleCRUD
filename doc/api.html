Welcome to my API Documentation

Get all todos Get a todo Create a todo Update a todo Delete a todo
Note: All things of this API are performed with Postman

Get all todos

To get all todos from my API:

https://todo-crud.up.railway.app/todos

Simple response

[
  {
      "_id": "63a698cb3a772316696921fb",
      "title": "Running 20km",
      "status": 2
  },
  {
      "_id": "63a6b4998f6ec0f3c3ec79e7",
      "title": "Running 20km",
      "status": 2
  }
]

Get a todo

To get a specific todo:

https://todo-crud.up.railway.app/todo/{id}

Where {id} is the todo's id that you've known

Example: id: 63a698cb3a772316696921fb

Response

{
    "_id": "63a698cb3a772316696921fb",
    "title": "Running 20km",
    "status": 2
}

Error response

{
    "message": "mongo: no documents in result"
}

Create a todo

Requirements: All fields of the request body must not be empty

This is request body

{
    "title": "Becoming a tech leader",
    "status": 1
}

API's url

https://todo-crud.up.railway.app/todo

Simple response

{
    "InsertedID": "63a6ce111c3ed53d7632c73e"
}

Update a todo

To update a todo, you must have a specific id of the todo and the body of the request where you want to modify

Body

{
    "title": "Becoming a tech leader",
    "status": 2
}
https://todo-crud.up.railway.app/todo/{id}

Example: id: 63a698cb3a772316696921fb

Response

{
    "message": "Successfully updated!"
}

Delete a todo

To delete a todo you must have its specific id

https://todo-crud.up.railway.app/todo/{id}

For example: id: 63a698cb3a772316696921fb

Response

{
    "message": "Successfully deleted!"
}
